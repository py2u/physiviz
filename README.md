# Physiviz

面向高中物理课堂的交互式可视化教学平台，采用原生 HTML、CSS、JavaScript 和 Three.js 构建。

## 内置课程

- 电场线与合等势面：支持点电荷放置、编辑、撤销、典型场景预设、总电势等值面，以及双电荷连线/中垂线上的电场和电势定量剖面。
- 简谐运动与单位圆：同步展示圆周运动投影、振子运动和位移—时间图像。

## 本地运行

项目无需构建。请在项目目录启动静态文件服务器，例如：

```powershell
python -m http.server 8765
```

然后访问 `http://127.0.0.1:8765/`。

Three.js 已保存在 `vendor/` 目录，可在无外网的课堂环境使用。

## 浏览器支持

推荐使用最新版 Chrome、Edge 或 Firefox。三维课程需要浏览器支持 WebGL。

## 部署

仓库通过 GitHub Pages 从默认分支根目录发布。
